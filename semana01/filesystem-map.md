# Mapa del Sistema de Archivos Linux
## / 
Directorio raiz del sistema. Todos los demas directorios nacen aqui.
## /home
Contiene los directorios personales de los usuarios.
## /etc
Contiene archivos de configuracion del sistema.
## /var
Contiene archivos variables como logs y cache.
## /bin
Contiene comnados esenciales como ls, cp, mv.
## /usr
Contiene programas y librerias de usuario.
## /opt
Software adicional instalado manualmente.
## /tmp
Archivos temporales que se borran al reiniciar.
## /dev
Dispositivos del sistema representados como archivos.
## /proc
Informacion del sistema y procesos en ejecucion.
## Conclusion
La estructura del sistema Linux organiza los archivos de forma jerarquica la raiz (/).
