# Semana 05 - Bash Scripting

## Descripción
El script sysinfo.sh genera un reporte del estado del sistema.

Incluye:
- Información general del sistema
- CPU
- Memoria RAM
- Uso de disco
- Procesos activos

## Uso

./sysinfo.sh

Opciones:

--cpu   Mostrar solo CPU
--mem   Mostrar solo memoria
--disk  Mostrar solo disco
--proc  Mostrar solo procesos
--help  Mostrar ayuda
--version Mostrar versión
