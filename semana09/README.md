# Semana 09: Gestión de Procesos

## Descripción

`monitor-procesos.sh` es una herramienta CLI en Bash para inspeccionar procesos del sistema, detectar procesos problemáticos, verificar procesos específicos, detectar zombies y generar reportes con timestamp.

## Uso

```bash
./monitor-procesos.sh [opciones] [proceso1 proceso2 ...]
