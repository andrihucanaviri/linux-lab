# Paquetes del Dev Stack

## Herramientas Esenciales

| Paquete | Comando | Versión mínima | Razón |
|---------|---------|----------------|-------|
| git | git | 2.x | Control de versiones del curso |
| curl | curl | 7.x | Descarga de recursos HTTP |
| wget | wget | 1.x | Alternativa a curl para descargas |
| vim | vim | 8.x | Editor principal del curso |
| tree | tree | 1.x | Visualización de directorios |
| htop | htop | 3.x | Monitoreo interactivo de procesos |
| jq | jq | 1.6+ | Procesamiento de APIs JSON |

## Entorno de Desarrollo

| Paquete | Razón |
|---------|-------|
| python3 | Scripting y automatización |
| python3-pip | Instalar librerías Python |
| build-essential | Compilar paquetes con extensión C |

## Utilidades del Sistema

| Paquete | Razón |
|---------|-------|
| unzip | Extraer archivos .zip |
| tmux | Sesiones de terminal persistentes |
| shellcheck | Validar scripts Bash |
| net-tools | netstat para diagnóstico de red |
